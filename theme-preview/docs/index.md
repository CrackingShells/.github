# CrackingShells Theme Preview

Welcome to the **CrackingShells** brand theme showcase. This dummy site lets you evaluate the **Egg Shell** (light) and **Hatch Night** (dark) themes before applying them to real documentation.

Toggle between themes using the brightness icon in the top-right corner of the navigation bar.

---

## About the Brand

The CrackingShells visual identity is built around two complementary color stories:

| Palette role | Light ("Egg Shell") | Dark ("Hatch Night") |
|---|---|---|
| Nav / header | Forest green `#3D5148` | Deep green `#2A3D32` |
| Page background | Warm cream `#F7F3EA` | Near-black `#111D18` |
| Body text | Deep dark green `#1D3328` | Off-white `#E8DFC8` |
| Links & accents | Warm amber `#D4952A` | Golden amber `#E8B84B` |
| Code blocks | Parchment `#EDE5CE` | Dark surface `#1A2B22` |

## Feature highlights

- Full `light` / `dark` toggle baked into the nav bar
- All Material component families (admonitions, tabs, tables, code) restyled
- Logos swap automatically on theme toggle via `overrides/main.html`
- Footer uses the darkest brand green for strong grounding
